# 🏆 ทีมสุขภาพดีแม่อุ๋ม

Web App สำหรับติดตามแต้มและความคืบหน้าของแพลน "ค่าจ้างเลิกเบียร์" ของแม่อุ๋ม

## 📱 Telegram Mini App
เปิดผ่าน Telegram Bot เพื่อดูสถานะปัจจุบัน:
- 🔥 Streck ปัจจุบัน
- ⭐ แต้มสะสม
- 🎁 รางวัลที่ใกล้ถึง
- 🏅 ความสำเร็จ
- 📅 ประวัติ check-in

## 🔧 วิธี Deploy
1. Fork repo นี้
2. เข้า Settings > Pages
3. เลือก branch `main` → `/ (root)`
4. Save — เสร็จ!
